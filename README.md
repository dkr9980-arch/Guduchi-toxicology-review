About the Study
This repository contains my published review paper on *Tinospora cordifolia* (Guduchi), a widely used medicinal plant in Ayurveda.

The study critically evaluates:
- Pharmacological properties  
- Toxicological profile  
- Hepatoprotective and immunomodulatory effects  
- Safety concerns and herb-drug interactions

Guduchi, Tinospora cordifolia, Toxicology, Ayurveda, Immunomodulation, Hepatoprotection
